# RemoteTruco  

Projeto desenvolvido para a disciplina de **Redes de Computadores** da **UERGS**.  
O objetivo é implementar um jogo de **Truco online** onde até **4 jogadores** podem se conectar a uma sala e jogar seguindo as regras declaradas em **regras-truco.pdf**.

## Descrição do Projeto  
O **RemoteTruco** será dividido em duas partes principais:  
- **Servidor:** Responsável por gerenciar as salas de jogo, receber conexões e garantir a comunicação entre os jogadores.  
- **Cliente:** Interface com a qual o jogador interage, enviando jogadas e recebendo as ações dos outros participantes.  

A comunicação entre cliente e servidor será feita por meio de **sockets**.  

As regras do jogo serão baseadas no documento **regras-truco.pdf** fornecido pela disciplina.  

## Funcionalidades Principais  
- Criar e gerenciar salas de jogo para até 4 jogadores.  
- Conexão entre cliente e servidor via sockets.  
- Interface para jogadas e exibição das cartas.  
- Implementação das regras de Truco conforme o pdf.
